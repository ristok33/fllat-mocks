# Fllat Mocks

Static HTML mockups for the Fllat no-deposit booking flow.

## Pages

| File | Description |
|------|-------------|
| `step1-rooms.html` | Room listings (London) |
| `step2-room-detail.html` | Room detail — Flex Plus, Canary Wharf |
| `step3-book.html` | Booking — select dates & deposit option |
| `step4-payment-plan.html` | Payment plan summary |
| `step5-review.html` | Review booking details |
| `step6-reserve.html` | Reservation confirmation |

## Usage

Open any HTML file directly in a browser:

```
open step1-rooms.html
```

Navigate through the flow using the links/buttons on each page.
